# Pharmacy-Sales-Profitability-Analytics

📌 Project Overview

This project analyzes a European pharmacy chain distributor operating across multiple countries. The objective was to build an interactive Power BI report to help stakeholders understand sales performance, profitability drivers, and regional contribution.

🎯 Business Objectives

Analyze how revenue and margin vary across countries and regions

Identify outperforming and underperforming pharmacies within regions

Compare Urban, Suburban, and Rural performance

Evaluate product category and brand profitability

Assess promotion impact on revenue, margin %, and price per unit

Identify geographic patterns using map visualization

📈 Key Insights

Non-promoted sales contribute ~90% of total revenue with higher margin %.

Promotions reduce revenue per unit (~11% drop) and significantly reduce margin %.

Some regions show high revenue but lower profitability, indicating pricing pressure.

Certain pharmacies outperform regional averages, highlighting performance gaps.
